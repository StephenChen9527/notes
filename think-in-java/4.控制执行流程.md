### 控制执行流程
控制流程主要包含的关键字有：`if-else`、`while`、`do-while`、`for`、`return`、`break`以及`switch`。

Java中没有`goto`语句，但是还是保留了关键字。

##### true和false
条件语句使用条件表达式的真假来决定执行哪条分支。Java中的控制语句，只允许`true`和`false`两个值。

简单示例：
```java
    if(1==1){}
    boolean a=false;
    if(a){}
```
`==`为判断是否相等，结果是一个`boolean`值，因此用于`if`后的括号中。
